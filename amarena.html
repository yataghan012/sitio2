    We created a website for Amarena Bakery, a bakery located in Córdoba, Argentina

01

CONCEPT

Since the bakery is located in Córdoba and targets customers looking for a special, artisanal experience, we aimed for a warm yet refined atmosphere rather than the sterile look of a typical pastry shop — drawing inspiration from the aesthetic of European patisseries. For the key visual, we photographed a carefully arranged selection of the bakery's handcrafted products and ingredients, conveying both craftsmanship and attention to detail while achieving a magazine-worthy, editorial feel. The main color palette was derived from the tones found inside the bakery itself, subtly deepened, so that the website's visual identity feels like a seamless extension of the physical space.

02

DEVELOPMENT

For the development of the website, we kept the technical stack intentionally lightweight and dependency-free. Rather than relying on external plugins or a CMS, the entire site was built as a single, self-contained HTML file — keeping load times fast and the codebase easy to maintain. Scroll animations and interactive elements were handled with vanilla CSS and minimal JavaScript, and all assets are served statically via GitHub Pages, making deployment and future updates straightforward.
